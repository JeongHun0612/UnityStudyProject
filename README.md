# C# 기초 수업용 Unity 프로젝트

> 수업자료 Notion: [C# 기초 수업 정리](https://app.notion.com/p/372aa7065267808ea830d2859cd3cd42?v=372aa7065267807d98a4000c8e9c090d&source=copy_link)

이 저장소는 **C# 기초 수업 실습**을 위해 만든 Unity 프로젝트입니다.  
문법만 따로 배우는 데서 끝나지 않고, 변수, 조건문, 반복문 같은 기본 개념을 **간단한 게임형 실습**으로 익히는 것을 목표로 합니다.

## 프로젝트 목적

- C# 기초 문법을 Unity 환경에서 직접 적용해 보기
- 수업 시간에 다룬 개념을 씬 단위 실습으로 복습하기
- 작은 기능을 직접 만들며 프로그래밍 흐름에 익숙해지기

## 실행 환경

- Unity Editor `6000.4.5f1`
- Unity 기반 C# 기초 학습 프로젝트

## 실행 방법

1. Unity Hub에서 이 저장소의 `UnityStudyProject` 폴더를 엽니다.
2. 프로젝트가 열리면 `Assets` 아래의 실습 씬을 선택합니다.
3. Play 버튼을 눌러 각 예제를 실행하고 동작을 확인합니다.
4. 스크립트를 수정하며 수업 내용에 맞춰 직접 실습합니다.

## 이런 분에게 적합합니다

- C# 문법을 처음 배우는 수강생
- 조건문과 반복문을 실제 예제로 익히고 싶은 학습자
- Unity를 활용해 프로그래밍 기초를 재미있게 시작하고 싶은 분

## 비고

이 프로젝트는 **강의 및 실습 중심 저장소**이므로, 완성된 상용 게임 프로젝트 구조보다는  
학습 단계에 맞춘 단원별 정리와 예제 실습에 초점을 두고 있습니다.
